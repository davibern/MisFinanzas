# Mis Finanzas

Proyecto para gestionar mis finanzas personales.

## Objetivo

El objetivo es importar la lista de movimientos bancarios y tener un resumen de mis finanzas.

## Características

- Importar movimientos bancarios desde un archivo XML.
- Resumen de ingresos y gastos.
- Gráficos de ingresos y gastos.
- Exportar el resumen a un archivo CSV.
