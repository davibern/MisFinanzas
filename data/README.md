# Directorio de Datos

Este directorio contiene los datos utilizados por la aplicación para representar la estructura de los ficheros y los datos esperados que deben tener en producción.

## Ficheros de Ejemplo (`.parquet.example`)

Encontrarás varios ficheros con la extensión `.parquet.example` en este directorio. Estos ficheros se proporcionan únicamente como referencias para mostrar la estructura y el formato esperado de los datos.

⚠️ **IMPORTANTE: Datos Ficticios** ⚠️

Hago especial hincapié en que la información contenida en todos los ficheros `.parquet.example` **son DATOS COMPLETAMENTE FICTICIOS**. 

Estos datos **NO representan la realidad**, no corresponden a ninguna situación financiera real, y han sido generados artificialmente de forma aleatoria con el único propósito de servir como datos de prueba y demostración. No deben ser tomados en cuenta bajo ninguna circunstancia como datos reales.
